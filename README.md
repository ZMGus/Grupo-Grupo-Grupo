# Grupo-Grupo-Grupo
Este repositorio reúne los proyectos del semestre asociados al ramo Taller de Programación


## Índice de proyectos
- **[01 — Diseño de la Red de Distribución](proyectos/01-diseño-red-distribucion/README.md):** localización de plantas y asignación de flujos multi-año con Gurobipy (Python), visualización en mapa con Cartopy y link al video.
- **[02 — Diseño de Proyecto](proyectos/02-Diseño-de-Proyecto/README.md):** Localización de adultos mayores dentro de hogares de cuidado de adultos mayores: diagrama en draw.io más prototipo en Unity..
- **[03 — Modelo-Prediccion-Femboy](proyectos/03-Modelo-Prediccion-Femboy/README.md):** Proyecto de Machine Learning para clasificar perfiles de GitHub como "femboy" basándose en métricas de interacción y calidad de código mediante un modelo Random Forest.
